📊 工程造价审核数据校验

基于 Word 会议议题文件（.docx）与 Excel 明细表（.xls/.xlsx）自动比对项目信息、金额、核减率等关键字段，输出详细差异报告。
⚡ 仅支持 .docx (强烈推荐)
第三行作为列名，第四行起数据
请上传文件并点击「开始对比校验」

⚙️ 编辑匹配规则
# 显示名称 Word 正则表达式 Excel 列名 Key 匹配类型 转换系数 操作
💡 提示：修改后自动保存到浏览器本地存储（localStorage）。支持导入/导出 JSON 配置文件。
📐 转换系数说明：Excel值 × 转换系数 = 比较值。例如：Excel单位为万元，Word单位为元，系数填 10000。
📋 Excel列名中的换行符会自动去除，填写时无需包含 \n。
⚙️ 比对规则说明：Word金额单位（元） vs Excel单位（万元）自动换算；百分比字段（核减率/误差率）Excel存储为小数（如0.0523）自动乘100与Word百分比数值比较。
📌 项目名称匹配依据“项目基本情况”分割段落，正则提取各字段；Excel第三行Key严格对应原模板列名。
